Код буде виглядати наступним чином: 
y = [0, 0]
c = [0, 0]
m = [0, 0]
s = [0, 0]

for i in range(len(seaside)):
    for j in range(len(seaside[i])):
        if seaside[i][j] == 'Y':
            y = [i, j]
        elif seaside[i][j] == 'C':
            c = [i, j]
        elif seaside[i][j] == 'M':
            m = [i, j]
        elif seaside[i][j] == 'S':
            s = [i, j]
По-перше, ви створюєте змінні зі списками координат та присвоюєте їм значення [0, 0]. Після чого за допомогою циклу перевіряєте кожен елемент двовимірного масиву і записуєте його координати у відповідну змінну.
Як, знаючи координати, мені розрахувати загальну кількість кроків?
Вам потрібно визначити різницю координат між вашим місцеположенням та розташуванням однієї з речей, а потім обрати максимальне із двох значень - різницю по горизонталі та по вертикалі. Це можна зробити за допомогою функції max(). Після чого вам буде потрібно виконати аналогічні дії для двох інших речей і підрахувати загальну суму трьох відстаней.
Мені потрібен приклад коду, щоб розібратись.
Для зручності, ви можете описати дану частину рішення в окремій функції: 
def dist(a, b):
    return max(abs(a[0]-b[0]), abs(a[1]-b[1]))
де a та b - списки з координатами вашого місцеположення і координати однієї з речей.
